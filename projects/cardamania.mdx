Cardamania is the flagship project of Left-Sided Studios — a TCG battle game built around Commander-style deckbuilding inspired by Magic the Gathering. The game pits players against each other in Academy Battles, representing their faction as they compete for dominance. The current target release window is **2028**.

## Gameplay overview

Cardamania is built around three core pillars:

- **Commander-style gameplay** — Deep tactical deckbuilding with dedicated commander cards that define your playstyle. Every match involves carefully constructed decks competing for board control.
- **Online multiplayer** — Academy Battles are fought against real opponents over dedicated servers. The multiplayer experience is the central focus of the game.
- **Academy system** — Players align with an Academy faction, representing it throughout ranked matches and progression.

## Development roadmap

<Steps>
  <Step title="Core Deck Engine — 2024">
    The foundational card and rules engine was completed in 2024. This covers card parsing, turn structure, win conditions, and local game logic.
  </Step>
  <Step title="Multiplayer Netcode — 2025">
    Dedicated server infrastructure and networking stability for Academy Battles. This milestone is currently in progress and is the primary focus of the team.
  </Step>
  <Step title="V1.0 Launch — 2028">
    Full public release on PC. Includes all Academy factions, ranked multiplayer, and the complete card set for the base game.
  </Step>
</Steps>

<Note>
  The release window for Cardamania was shifted to **2028** (from an earlier target) to ensure the multiplayer infrastructure meets the standards required for competitive Academy Battles. The Commander Engine demands networking stability comparable to established TCG titles — latency issues during tactical planning would undermine the core experience. This extension allows the team to double down on balance testing and server infrastructure before launch.
</Note>
