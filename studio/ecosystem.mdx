Left-Sided Studios is not a single monolithic studio — it operates as a parent organisation overseeing a network of subsidiary labels and strategic partnerships. Each label has a focused creative mandate, letting the collective tackle horror, RPG, action, simulation, and media production simultaneously without diluting the quality of any single output.

## Organisational structure

Left-Sided Studios sits at the top of the hierarchy. Below it are four subsidiaries and one shared production unit, aligned alongside two strategic and creative partners.

```
Left-Sided Studios (Parent Studio)
├── ANdE Studios          [Founder Imprint — Vermetra]
├── Gotch-ya Studios      [Founder Imprint — DaRealSansYT]
├── Bomb Banana Studios   [Founder Imprint — RocketBlasts]
├── Endgame Studios       [Official Subsidiary]
└── Skullix Media Group   [Production Unit — Joint Venture]

Partners
├── Citadel Studios       [Strategic Partner]
└── Geekza Productions    [Creative Partner]
```

## Label types explained

| Type | Description |
|---|---|
| Founder Imprint | Personal creative label owned and operated by a single co-founder |
| Official Subsidiary | A fully integrated label within the LSS structure |
| Production Unit | A service-oriented unit supporting media delivery across labels |
| Strategic Partner | An external studio aligned with LSS on shared development goals |
| Creative Partner | An external studio collaborating on creative and storytelling projects |
